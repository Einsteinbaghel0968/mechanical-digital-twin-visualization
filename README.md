# Mechanical System Digital Twin Visualization

This project demonstrates a **Digital Twin model for a mechanical system** using simulated sensor data and machine learning techniques.

The notebook analyzes machine behaviour through multiple analytical visualizations and monitoring techniques.

---

## System Architecture

The physical system modeled in the digital twin:

Motor → Pump → Valve → Hydraulic Line → Actuator

Sensors monitor:

- RPM
- Pressure
- Flow
- Temperature
- Vibration

---

## Project Workflow

1. Sensor Data Simulation
2. Time Series Visualization
3. Correlation Analysis
4. Frequency Analysis (FFT)
5. PCA Machine Health Space
6. Anomaly Detection
7. Sensor Interaction Network
8. Digital Twin System Architecture

---

## Visualizations

The project includes multiple analysis plots:

- Sensor time series monitoring
- Correlation heatmap
- FFT vibration spectrum
- PCA machine health space
- Anomaly detection visualization
- Sensor interaction network
- Mechanical digital twin architecture diagram
